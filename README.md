# WEB. Laboratory 3

### Usage
```
git clone git@github.com:urasha/WEB_JSF.git
cd WEB_JSF
docker compose up --build
```
and go to [http://localhost:8080/web3](http://localhost:8080/web3/) when WildFly starts

### Stack
- JavaServer Faces
- JPA + Oracle
- Maven
- WildFly + Docker
